# initialize project

1. Create requirements file with all requirements need for you project:
    - django==4.1.7
    - djangorestframework==3.14.0

2. Execute command in terminal: "pip install -r requirements.txt"

3. Run command to initialize django project: "python -m django startproject tasks ."

4. Run command to create django web api: "python manage.py startapp tasks_api" 

5. Run djnago server with command: "python manage.py runserver"

Important follow these link:
    start new project: https://radixweb.com/blog/create-rest-api-using-django-rest-framework
